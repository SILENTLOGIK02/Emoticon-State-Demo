# Emoticon State Demo

🔹 Description:
This is a simple Python program that demonstrates the use of global variables and functions by changing an emoticon based on program state.

🔹 Features:
- Uses a global variable to store emoticon state
- Demonstrates function calls and shared variables
- Shows how program state can change during execution
- Prints dynamic messages based on variable updates

🔹 Technologies:
- Python

🔹 Purpose:
Created to practice global variables, function behavior, and understanding how state changes affect program output.
